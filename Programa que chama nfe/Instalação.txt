COMO INSTALAR

1 - Compartilar a unidade do Servidor "onde os programas do parceiro transmissor foram instalados"

2 - Mapear essa unidade no computador que vai gerar os CTes como unidade Z:

3 - Copiar o programa GBCTe para qualquer pasta desse computador que vai gerar os CTes.

4 - Criar um atalho no Desktop apontando para o programa GBC

5 - Execupe o aplicativo.
